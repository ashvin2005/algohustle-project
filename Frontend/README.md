# AlgoHustle Frontend

React frontend for AlgoHustle - a competitive programming progress tracker.

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Chart.js
- React Router
- Axios

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file:

```
VITE_API_URL=http://localhost:5001/api
```
